# 💬 카카오톡 오픈프로필 셋팅 가이드

오픈프로필을 만들면 사이트에서 클릭 한 번으로 카톡 1:1 채팅이 열립니다. 개인 카톡과 분리되어 운영도 깔끔해져요.

---

## 🛠️ 만드는 방법 (5분)

1. 카카오톡 앱 → **채팅 탭**
2. 우상단 **+ 버튼** → **오픈채팅**
3. **오픈프로필 만들기** 선택
4. 아래 정보 입력 (다음 섹션 참고)
5. URL 발급 받음 (예: `https://open.kakao.com/me/andrew-ireland`)
6. 그 URL을 Andrew에게 알려주면 → HTML 사이트 카톡 버튼이 즉시 활성화

## ✏️ 입력할 정보 — 추천안

### 1. 이름 (Name)

추천 옵션 — 검색되기 쉽고 정체성 명확:

| 후보 | 길이 | 평가 |
|------|------|------|
| `🇮🇪 Andrew의 아일랜드 동행` | 14자 | ⭐⭐⭐ 브랜드 명확 |
| `서현 Andrew · 더블린 동행` | 13자 | ⭐⭐⭐ 이름 + 서비스 |
| `🎼 서현 Andrew · 아일랜드 동행` | 16자 | ⭐⭐ 음표 = CMfE 연결 |
| `Andrew 서현 · Beautiful Ireland` | 25자 | ⭐ 영문 우선 |
| `🇮🇪 Beautiful Ireland with Everyone` | 30자 | ⭐⭐ 브랜드만 |

**🏆 1순위 추천**:
```
🇮🇪 Andrew의 아일랜드 동행
```

이유:
- "Andrew" = 너의 정체성
- "아일랜드 동행" = 검색 키워드 + 무엇을 하는지 명확
- 🇮🇪 이모지 = 시각적 임팩트
- 한국어 손님이 카톡에서 검색해 찾기 쉬움

```

이유:

**🥈 2순위 (서비스 강조 원하면)**:
```

## 🔗 URL 받은 후 다음 단계

URL을 받으면 (예: `https://open.kakao.com/me/abcd1234`):

1. 사이트 푸터 카톡 버튼이 클릭 즉시 채팅 열리도록 자동 활성화 (HTML 업데이트)

3. 인스타 바이오·네이버 블로그 등에도 동일 URL 게시 가능

→ **오픈프로필 URL을 알려주면 v11에서 즉시 적용**.
